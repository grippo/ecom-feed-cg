# W3C E-Commerce Product Feed CG

Universal machine-readable product feeds.

## Install & Validate

```bash
npm install
npx ecom-feed-validate examples/valid-single.json
```

## Run Demo Server

```bash
node implementations/node/server.js
```

## Spec

* Core Schema
* JSON-LD Context
* OpenAPI

## Test

```bash
npm test
```
